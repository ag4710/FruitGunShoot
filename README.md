# FruitGunShoot
2023 ISHS 라이빗 유니티 게임

### 활용 Asset
https://assetstore.unity.com/packages/vfx/particles/cartoon-fx-remaster-free-109565

### 게임 설명

##### 게임 목표: 60초동안 과일을 맞춰 점수를 얻어라

+ start 게임 시작
+ 60초 동안 과일을 맞추면 된다.
+ 모든 과일을 맞추면 과일이 초기 상태로 돌아간다.
+ 총알은 32개, 과일 개수도 32개
+ 다 맞추면 10000점
+ Exit 누르면 게임 종료

##### 게임 조작: 마우스로 클릭

##### 게임 점수:
+ Orange: 200
+ Apple: 300
+ Lemon: 450
